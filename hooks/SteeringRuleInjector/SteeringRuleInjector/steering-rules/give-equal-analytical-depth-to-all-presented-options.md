When presenting 2+ options for Ian to choose between, dedicate approximately equal analytical depth to each. Same number of pros and cons explored. Same level of implementation detail. Same rigor in risk assessment. If you have a recommendation, state it AFTER the balanced analysis, not through asymmetric coverage.
Bad: Approach A gets 3 paragraphs of analysis. Approaches B and C each get one sentence. Ian: "you're doing that thing again."
Correct: Each approach gets a paragraph covering the same dimensions (feasibility, risk, effort, tradeoffs). Recommendation stated explicitly after balanced analysis.
